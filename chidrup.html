FuelSmart City is a fuel station management platform for a 1300 sq km coastal metropolis (~18 million people), focusing on optimal station placement, inventory planning, and routing to reduce travel time, congestion, and emissions.

Business Model Snapshot

Value Proposition

Plan locations, capacities, and fuel/EV inventory for each station so that users are never far from refuelling while operators maximize utilization.

View Architecture

Revenue Streams

Software subscription for operators, analytics services, route optimization for fleets, and premium green-energy routing for EV users.

SDG Alignment

Supports cleaner and more efficient energy use and more sustainable cities by cutting unnecessary trips and fuel wastage.

Core Techniques & Principles

Algorithms and Principles in Fuel Management
Category Chosen Approach Role in System
Technique Greedy + Dynamic Programming Greedy for assigning vehicles to nearest feasible stations and for quick placement decisions; DP for multi-period inventory planning and cost minimization.
Principles Partitioning, Shortest Path Trees, Space–Time Trade-off Partition city into zones, build shortest path trees on road network, and use precomputed routes and demand profiles to answer queries fast.
Sorting Quick Sort / Merge Sort Sort candidate sites by traffic or demand; sort routes and stations by distance or load for greedy choice.
Searching Rabin–Karp / Brute Force Pattern checks in logs and configuration strings; small searches use simple brute force.

System Architecture & Flow

City Partition Graph

The 1300 sq km region is partitioned into zones (coastal, urban core, peri‑urban, highway belts). Each zone has a set of candidate station nodes and demand points connected by weighted road edges.

Shortest Path Trees

For each zone, shortest path trees from stations to demand clusters give travel time and distance, used by greedy routing and DP inventory decisions.

Optimization Loop

Greedy placement picks high-traffic nodes first; DP refines daily inventory and supply plans, trading off stock-out risk vs holding cost.

C++ Sketch – Fuel Station Planning Core

#include <bits/stdc++.h>
using namespace std;

// Demand node in the city
struct DemandPoint {
    int id;
    double x, y;      // coordinates (km)
    double demand;    // daily fuel demand
};

// Candidate fuel station
struct Station {
    int id;
    double x, y;
    double capacity;  // daily supply capacity
};

// Euclidean distance as a simple proxy for travel cost
double dist(double x1, double y1, double x2, double y2) {
    double dx = x1 - x2, dy = y1 - y2;
    return sqrt(dx*dx + dy*dy);
}

// Greedy assignment: assign each demand point to nearest station
void greedyAssign(vector<DemandPoint> &dem, vector<Station> &st) {
    for (auto &d : dem) {
        int bestId = -1;
        double bestCost = 1e18;
        for (auto &s : st) {
            if (s.capacity < d.demand) continue; // capacity check
            double c = dist(d.x, d.y, s.x, s.y);
            if (c < bestCost) {
                bestCost = c;
                bestId = s.id;
            }
        }
        if (bestId != -1) {
            cout << "Assign demand " << d.id
                 << " to station " << bestId
                 << " (distance ~" << bestCost << " km)\n";
        } else {
            cout << "No feasible station for demand " << d.id << "\n";
        }
    }
}

// Simple memoized cost function for daily supply planning
map<pair<int,int>, double> memoCost;

double dailyCostDP(int stationId, int day, int horizon,
                   double baseCostPerKm) {
    if (day >= horizon) return 0.0;
    auto key = make_pair(stationId, day);
    if (memoCost.count(key)) return memoCost[key];

    // Example recurrence: increased cost on later days (e.g., congestion)
    double todayCost = baseCostPerKm * (1.0 + 0.05 * day);
    double nextCost  = dailyCostDP(stationId, day + 1, horizon, baseCostPerKm);

    double res = todayCost + nextCost;
    return memoCost[key] = res;
}

int main() {
    // Sample data
    vector<DemandPoint> demands = {
        {1, 2.0, 3.0, 500},
        {2, 10.0, 8.0, 700},
        {3, 6.0, 15.0, 400}
    };
    vector<Station> stations = {
        {1, 1.0, 2.0, 1000},
        {2, 9.0, 9.0, 800}
    };

    // Sort stations by capacity (Quick/Merge via std::sort)
    sort(stations.begin(), stations.end(),
        [](const Station &a, const Station &b) {
            return a.capacity > b.capacity;
        });

    greedyAssign(demands, stations);

    // DP cost estimation for station 1 across 7 days
    double totalCost = dailyCostDP(1, 0, 7, 1.0);
    cout << "Estimated 7-day base travel cost for station 1: "
         << totalCost << "\n";
    return 0;
}
