Module 3 ‚Äì Tourism & Resort Booking

A. Booking Engine & Dynamic Pricing

This module manages resort occupancy, processes booking requests with priorities, and adjusts prices dynamically based on demand and occupancy levels.

How the Code Works

A segment tree tracks daily occupancy and checks availability efficiently. Incoming booking requests are stored in a priority queue so high-priority customers are served first. If capacity allows, the booking is accepted and revenue is updated. A simple demand-based pricing rule increases prices as occupancy rises.

Efficiency Analysis

ComponentTime ComplexitySpace
Segment TreeO(log n)O(n)
Heap (Bookings)O(log m)O(m)

B. Itinerary Planner

Plans optimal travel itineraries between multiple tourist locations using shortest-path and TSP heuristics.

How the Code Works

Points of interest are treated as nodes. A nearest-neighbour heuristic builds an initial tour, and 2-opt optimization improves it. For real maps, Dijkstra or A* is used to compute segment costs.

Efficiency Analysis

AlgorithmTimeSpace

C. Review/Text Analysis & Recommender

Analyzes customer reviews for keyword search and generates recommendations using text similarity.

How the Code Works

KMP efficiently searches phrases inside reviews. An inverted index maps keywords to documents. Reviews are converted into sparse TF vectors and cosine similarity measures how similar two reviews are.

Efficiency Analysis

TechniqueTimeSpace
KMPO(n + m)O(m)
Inverted IndexO(total terms)O(total terms)

// ================= BOOKING ENGINE =================
#include <bits/stdc++.h>
using namespace std;

struct Seg{
    int n; vector<int> t;
    Seg(int N){ n=1; while(n<N) n<<=1; t.assign(2*n,0); }
    void add(int idx,int val){
        idx+=n; t[idx]+=val;
        for(idx>>=1; idx; idx>>=1)
            t[idx]=max(t[2*idx],t[2*idx+1]);
    }
    int range_max(int l,int r){
        int res=INT_MIN;
        for(l+=n,r+=n; l<=r; l>>=1,r>>=1){
            if(l&1) res=max(res,t[l++]);
            if(!(r&1)) res=max(res,t[r--]);
        }
        return res;
    }
};

// ================= TEXT ANALYSIS =================
vector<int> build_lps(const string &p){
    int m=p.size(); vector<int> lps(m,0);
    for(int i=1,len=0;i<m;){
        if(p[i]==p[len]) lps[i++]=++len;
        else if(len) len=lps[len-1];
        else lps[i++]=0;
    }
    return lps;
}
